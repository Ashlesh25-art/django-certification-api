# Django Certification Management API

This project is a backend API built using **Django** and **Django REST Framework**.

It manages relationships between:

Vendor → Product → Course → Certification

Example:
Microsoft → Azure → Azure Fundamentals → AZ-900

## Technologies Used
- Python
- Django
- Django REST Framework
- drf-yasg (Swagger)

## API Endpoints

Vendors  
GET /api/vendors  
POST /api/vendors  

Products  
GET /api/products  
POST /api/products  

Courses  
GET /api/courses  
POST /api/courses  

Certifications  
GET /api/certifications  
POST /api/certifications  

## API Documentation

Swagger UI available at:

http://127.0.0.1:8000/swagger/
